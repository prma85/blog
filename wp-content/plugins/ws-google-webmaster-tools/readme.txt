Easily Verify Google Analytics & Webmaster Tools for your Wordpress websites & blogs. 

== Description ==   

Google Analytics & Webmaster Tools provides reports and data that can help you understand how different pages on your website are appearing in search results.

Google Analytics is a freemium web analytics service offered by Google that tracks and reports website traffic. 

Google Webmaster Tools (GWT) is the primary mechanism for Google to communicate with webmasters.

== Installation ==
1. Upload the entire `ws-google-webmaster-tools` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' -> 'WS Webmaster Tools' in WordPress.
3. Add your Google Analytics tracking code, Webmaster Tools Verification Tracking code to the Google Webmaster Tools settings.
